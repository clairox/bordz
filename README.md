# Bordz

This is a mock e-commerce project of an online skateboarding shop

## Features

-   Fully functional cart system
-   Purchases can be made with Stripe (only with card number 4242 4242 4242 4242. Real purchases can't be made)

## Site

Visit the site at https://bordz.vercel.app/

## Technologies

-   [Next.js](https://nextjs.org/)
-   [Stripe](https://stripe.com/)
